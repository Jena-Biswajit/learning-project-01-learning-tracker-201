## Java Data Types

**Primitive Data Types**

| Data Type   | Size (in bytes) | Range                                              |
|-------------|-----------------|--------------------------------------              |
| `byte`      | 1               |  -128 to 127.                                      |
| `short`     | 2               |  -32,768 to 32,767.                                |
| `int`       | 4               |  -2^31 to 2^31-1.                                  |
| `long`      | 8               |  -2^63 to 2^63-1.                                  |
| `float`     | 4               | ±3.40282347E+38 (7 decimal places)                 |
| `double`    | 8               | ±1.79769313486231570E+308 (15 decimal places)      |
| `char`      | 2               | 0 to 65,535 (unsigned Unicode characters)          |
| `boolean`   | 1 (or 1 bit)    | 1 bit ,Stores `true` or `false`.                   |
